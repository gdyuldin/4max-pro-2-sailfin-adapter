# Sailfin adapter for Anycubic 4MAX Pro 2.0

## Sailfin extrider

STL availbale here: https://github.com/CroXY3D/Sailfin-Extruder

Parts to print:
* Front.stl
* Lever.stl
* Mid.stl
* Rear.stl

## Modification of the Sailfin extruder:

* `STL/sailfin_mod-rear_mod.stl` - highter rear part. Might be useful with some parts.
* `STL/sailfin_mod-sailfin_mid_for_print_2.stl` - mid part without filament hole (should be drilled with 2mm bit using PTFE tube as a guide).
